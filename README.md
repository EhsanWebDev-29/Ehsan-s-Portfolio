# Ehsan's Personal Portfolio

A modern, responsive and fully interactive **Frontend Developer Portfolio**, built using **HTML, CSS and JavaScript**.  
This website showcases my skills, projects, and services with a clean UI, smooth animations and a mobile-first design.

## Features

### Interactive Hero Carousel  
- Auto-slide  
- Next / Prev controls  
- Indicators  
- Pause on hover  
- Smooth infinite sliding  

### Light / Dark Theme  
- Theme toggle button  
- Saves user preference using localStorage  
- System theme auto-detect

### Fully Responsive  
Works on:  
✔ Mobile  
✔ Tablet  
✔ Laptop  
✔ Large screens  

### Modern UI Components  
- Bento-style services grid  
- Animated project cards  
- Auto-scrolling skills section  
- Hamburger menu with animation  
- Smooth scrolling  
- Back-to-top button  

### Contact Form → WhatsApp Integration  
Form instantly sends message through WhatsApp with user details.

## Technologies Used

- **HTML5**  
- **CSS3 (Flexbox, Grid, Animations)**  
- **Vanilla JavaScript**  
- **localStorage API**  
- **Responsive Design**  
- **GitHub Pages**
